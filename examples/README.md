# 示例案例

这里保存开发过程中生成的真实报告案例。

## wechat-article-codex

主题为 `OpenAI Codex` 的微信公众号文章热点报告。

文件：

- `report.md`
- `report.html`

## wechat-channel-codex

主题为 `Codex` 的微信视频号热点报告。

文件：

- `report.md`
- `report.html`

原始运行时曾下载本地 MP4 到 `assets/`，以便 HTML 报告播放视频。示例目录中不包含 MP4 文件，原因是：

- 避免 GitHub 仓库体积过大
- 避免未授权分发第三方视频内容

真实运行并启用 `--download-video` 时，会在报告目录旁生成 `assets/video-*.mp4`。

## 隐私

示例不包含 API Key、原始 API 响应，也不包含带 Key 的播放链接。
